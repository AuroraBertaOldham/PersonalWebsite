Beacon Games

My indie game company that is currently building Everchanging.

Everchanging

A city-building RPG that I've been developing for several years now. I plan to release the game sometime in 2021.

EvEngine

A 2D C# game engine that I wrote for Everchanging. It only supports Windows and DirectX 11 at the moment. I may port it to various other platforms after I release Everchanging.

Verdure OS

A modern open-source operating system for x86_64 PCs that I'm writing in Rust. It is still in very early development, but I plan to have a full desktop environment in the future. I also hope to bring it to AArch64 once that is completed.

The Verdure Project

An organization that I'm working on establishing to manage Verdure OS and other open-source projects.

SharpFNT

A C# library for reading and writing AngelCode bitmap fonts in XML, text, and binary. It was originally part of the EvEngine, but I decided to separate it and release it as open-source. Since its release, it has received over 100,000 downloads on NuGet and has found use in other games and game engines.

FNTTools

A .NET Core command-line tool built off of SharpFNT. It is capable of converting and inspecting AngelCode bitmap fonts.

Yacht (Elementary)

A GTK3 based implementation of the public domain dice game designed specifically for Elementary OS. Postponed until Elementary OS 6 and Everchanging are released.
